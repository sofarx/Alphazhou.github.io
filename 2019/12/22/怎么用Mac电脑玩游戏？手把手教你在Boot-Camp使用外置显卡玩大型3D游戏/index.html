怎么用Mac电脑玩游戏？手把手教你在Boot Camp使用外置显卡玩大型3D游戏

发表于2019-12-22|更新于2019-12-23|game
|阅读量:

前言

在一台Mac电脑上面设置BootCamp 外置显卡，对于一部分人来说是即插即用的，但是对于不熟悉的人来说，可以是噩梦一场；最早的有雷电三代接口的Mac是只有集成显卡的，例如13寸的MacBook Pro、2018款Mac mini。有独立显卡的Mac电脑就会经常对系统提出更多的资源需求，而windows经常不会顺从的，这种情况下就会经常出现Error12错误（资源不足），甚至在Boot Camp外置显卡启动windows的时候卡在启动界面，出现win logo就死机的情况。
现在Apple官方支持外置显卡（macOS10.13.4之后）了，epgu.io的这个专题网站就做了很多努力来设置、维护一个可以使的外置显卡在Boot Camp win10正常使用的方法，发展几年以来，egpu的社区提供了很多解决方法，以下的设置指导是其中一部分的集合。

显卡切换

简要说一下在Windows系统里面的显卡切换；egpu.io论坛的成员sky11解释了在win10里面三种模式(AMD XConnect, Nvidia Optimus, and Microsoft Hybrid Graphics)。为了实现显卡自动切换工作，必要的先决条件是有intel集成显卡。AMD XConnect/Nvidia Optimus 在集成intel显卡的情况下外界Radeon/Geforce外置显卡的情况下，提供了一种在快速切换集成显卡的方法。但在没有集成显卡和只有独显的Mac电脑，你需要Win10 1803以上的系统版本。Win10 1803以上的系统版本提供了在每一个应用程序/游戏手动切换显卡的设置。因此，最理想的Boot Camp外置显卡的环境是：只有集成显卡的Mac，运行Windows 10系统版本为 1903 OS Build 18362.295 ；ps：笔者实际安装使用的是Win10 1909版。

在设置外置显卡的之前要知道的，是你的Mac电脑的雷电3接口通往主板PCIe控制器是有不同的负载能力的，有些接口负载能力不足就会搞到系统崩溃。egpu.io检查之后总结了可以在Bootcamp用外置显卡的电脑有以下：

13寸MacBook Pro的CPU是U系列处理器，因此没有直接由雷电3接口直连CPU的借口。所有的PCIe通道都是通过平台控制中枢（PCH）来管理。论坛的高手指出，2016&2017 13“Touch bar MacBook Pro右边的雷电3接口比左边的借口有更低的带宽。具体来说，是右边的接口PCI Express Root Port 9 – 9D18只有X2的速度，左边接口有满血的X4速度。2018*2019款型号的用了第八代的四核U系列处理器提供了更高速的通道，右边雷电3接口都有满血的X4速度的PCIe通道。下面是15寸MacBook Pro左和13寸MacBook Pro右的硬件设备带宽对比图；

哪台mac最适合bootcamp使用外置显卡？

论坛的高手对比发现，2018款Mac mini是少有的没有独显的桌面电脑，对于外置显卡用户来说是天大的好消息，特别是用boocamp来装win10的。处理器是65瓦的，雷电3接口直连cpu，4个雷电接口有2个达到x8满血速度，有两个达到x4速度，而且没有独显干扰，是最值得拥有的外置显卡首选电脑。

哪台mac最不适合用外置显卡？

而iMac产品系列要在Boot Camp使用外置显卡是最复杂的。因为是为了桌面使用设计，没需要做多显卡支持，所以苹果在imac固件里面关闭了intel的集成显卡。建议不要用imac来配置bootcamp的外置显卡了。

AMD eGPU vs Nvidia eGPU哪个系列显卡最适合？

最好的答案当然是Mac系统就用adm显卡，Windows就用Nvidia显卡。当然一个amd显卡统治两个系统也是可能的，不过要折腾一番；如果要二选一的话，如果加入绿队（Nvidia），就可以直接使用windows开机出现logo的时候热插拔启动，似乎没什么问题。如果是红队（adm），情况就会复杂点就要准备一个叫做 Error12错误解决大冒险游戏。以下是macOS官方支持egpu，而且可以在bootcamp都能用的显卡列表：

大内存分配

为了在win10运行的时候不严重影响手提其他功能正常运行，你所有要做的事目的就是给外置显卡分配足够的资源。2017款和以后的雷电3mac固件都有都支持一个就做大型内存分配的功能。2016年的MacBook Pro就耀专门按照一个指引去编译一个DSDT的文件来，加载在启动项来支持这个功能；
在实际操作中，这个编译dsdt文件的步骤超级复杂，而且我实际按步骤操作实践是失败的，而且搞到严重死机要重装。 因为我的显卡是Nvidia的，不编译这个也能顺利运作。

低配macbookpro的设置

低配的MacBook Pro只有左边两个雷电3接口，应该会自动为接口获取到足够的带宽资源。如果出现 error12错误，在windows设备管理器禁用一些不必要的硬件，比如摄像头，就可以解决问题了。

热插拔外置显卡

只有少数的Mac电脑可以在Boot Camp windwos启动logo出现的时候插入雷电3线，然后可以识别到电脑是没问题的，论坛里面大部分成功案例都是Nvidia的显卡。AMD显卡大部分出现Error12错误提醒。
最可靠的方法还是冷接入，关机之后在插线操作。以下是推荐的步骤：

操作前，先了解了解背后的原理

步骤一：安装win10，并且设置系统还原点；

现在是19年的12月份，笔者已经在用win10的1909版win，微软提升了处理雷电3设备的热插拔检测，尽管如此，以往的经验是正常使用外置显卡过程中拔掉雷电3线，系统一定会死翘翘，所以不建议大家使用途中拔线；

在微软官网下载了最新版的win10版本iso文件之后，用Mac系统使用bootcamp来安装windows，预留40-50GB左右空间，系统会自动分区安装；

论坛有些指引使用win-to-go来将win10安装在外置硬盘，但是我们不建议这样做，因为，1现在新版的win版本已经放弃更新win-to-go的更新，2会导致外置显卡安装过程起来会有不可预料的问题。硬盘空间不够的话，游戏和应用装在外置硬盘是没问题的，系统就不要了。

安装完win10，在资源管理器找到bootcamp的临时加载盘，找到bootcamp工具驱动安装包，一键自动安装驱动。
安装过程系统会自动检测驱动自动安装，但很多都是旧的。egpu论坛会推荐这个时候使用 DDU 工具来删除Radeon Pro系列显卡的驱动，这个工具同事都可以关闭windows显卡自动安装程序。而且可以精确控制用什么版本的驱动。如果你的mac有radeon 独显卡，而且打算用radeon的外置显卡，可以从这个网站（bootcampdrivers.com）下载修改版的驱动来控制独显和外置显卡的运作。
记住，Apple自己在bootcamp不提供外置显卡的支持服务，所以以下步骤配置都可能导致mac电脑的启动问题。强烈建议用windows自带的系统保护机制，在几个关键步骤建立系统还原点，死机的时候还原一下就可以了。

步骤二：制作EFI启动盘；

在启动的过程，如果不是启动macOS自己的系统时候，mac的固件检测到有外置显卡的情况下，就会尝试关闭集成显卡。导致会经常出现死机情况。最明显的症状就是卡在windowsLogo，甚至黑屏。要跨过这个障碍，我们就要靠 apple_set_os.efi 来“欺骗”电脑，在启动过程以为自己是在启动macOS系统，实际上我们是引导来开win系统了。这个启动引导器可以存在U盘或者mac分一个区来做启动管理。
强烈建议用 Goalque 大神的方法 automate-eGPU EFI ,这个启动盘包含了上面提到的apple_set_os.efi启动文件。大神提供了一个清晰的建议，制作一个U盘来启动，这样系统的内核就会完好无缺。
第二种方法是使用 rEFInd 这个方法我试过，不建议使用了。

原文说到 有独显的MacBook Pro 15寸还要安装一个bat命令，用于关闭独立显卡的功能，这里就不翻译了，因为我没用上哈。
In the 2016 and 2017 15″ MacBook Pro , we want to attach the iGPU to the internal display so that we can use eGPU internal display acceleration and disable the dGPU if needed. This is where 0xbb’s integrated.bat comes in. The purpose of 0xbb’s GPU-switch script is to assign a particular GPU to the internal display at the next boot. We want to run integrated.bat in Command with Administrative privilege to execute iGPU attachment to the internal display. Upon the next and subsequent restarts, Windows will use the iGPU to power the Mac internal display.  Microsoft Visual C++ 2013 may be required to run this script successfully. Users of 2018 & 2019 MacBook Pro with a Radeon Pro dGPU (both 15-in & 16-in) are encouraged to visit the build guides . This is due to Apple completely disabled the Intel iGPU in these Mac laptops. Force-activation and internal display attachment of the iGPU may cause black screen issue.

步骤三：解决Eroor12错误；

最后而且最重要的步骤是客服出现error12错误提示（egpu带宽资源不足）这是经常发生的问题，最常见到的是有独显的mac搭配AMD Radeon外置显卡。这个错误不会自动弹出来。而是在设备管理器，在外置显卡的地方出现黄色禁用标志，双击打开里面会看到错误12提示。

最简便的变通做法是禁用PCIe一些硬件应用来释放足够的带宽资源来让给外置显卡用。每台Mac的PCIe分配都是不同的，但步骤类似。

13寸低配版MacBook Pro，禁用iSight摄像头的PCIe桥；这样应该可以解决问题了。

13寸touch bar版 除了关闭摄像头之外，还要多一步；关闭右边的雷电3接口控制器，大部分人是禁用了PCI Express Root Port #9 – 9D18 这个接口，然后用左边的雷电3接口 靠近tab键的那个接口来链接外置显卡。我亲自对比过左边两个接口，上边的明显卡，下面的游戏画面非常顺畅！

15寸的操作，我就不翻译了：
The 15″ MacBook Pro is rather challenging to allocate resources for the external graphics card. By disabling the x16 PCIe controller – 1901 to the discrete Radeon Pro GPU, we found Windows likely has enough resources for an external GPU. From eGPU.io members’ experience, Windows has a harder time with AMD cards. When we tried AMD eGPU, we needed to also disable the PCI Express x8 Controller – 1905 that connects to the left Thunderbolt 3 ports. External GPU works reliably using the right-side ports that connect to the PCI Express x4 Controller – 1909. Again 2018-2019 15″ & 16″ MacBook Pro have a completely disabled iGPU and would need to use Windows Hybrid graphics instead of AMD XConnect/Nvidia Optimus through the Intel iGPU.

安装详细操作步骤

一. Win10 iso和 DDU显卡驱动删除

下载链接： Windows 10 ISO and DDU

  1. 在macOS用Boot Camp转换助理安装win10 ISO文件；
    选择iso文件，分配好windows分区的空间，系统会自动安装。
    安装完win10，在资源管理器找到bootcamp的临时加载盘，找到bootcamp工具驱动安装包，一键自动安装驱动。

  2. 安装DDU工具清除默认的显卡驱动，关闭自动显卡安装程序
    右键使用管理员权限运行软件，选择amd驱动，卸载并重启。在设置选项里面关闭win10自动升级驱动。（13寸的用户应该不用这一步也行，因为没有独显没有安装amd的驱动）

  3. 制作一个windows系统还原点以防止出错无法启动
    在windows左下角搜索栏输入 系统还原 打开设置，选择创建一个系统还原点。确认保护是打开状态，然后创建还原点。

二、 制作启动盘和

下载链接 automate-eGPU EFI and  Intel graphics drivers

  1. 下载并复制 automate-eGPU EFI 启动文件，放在U盘或者设置一个系统分区；
    磁盘工具：格式化分一个32mb左右的分区来放这个启动引导文件；格式是ms-dos（FAT）格式；将efi文件复制在里面。

  1. 开机长按option键，选择通过automate-egpu引导盘（黄色的efi盘）启动
    到了下面的界面，按Q键，然后按下⬇️键，选择windows boot启动。最下面的文字 undetected egpu的提示作者说可以忽略。

  2. 在windows设备管理器确认集成显卡并安装intel的驱动
    启动windwos之后，在设备管理器查看，在显示设备 看到微软基础显示适配器，就继续安装驱动；

    没有独显的mac，比如13寸macbookpro应该不用做这一步骤，因为集成显卡的驱动在bootcamp安装的时候就安装了驱动了。

  3. 创建windows系统还原点；
    接下来就会接入外置显卡，操作前按上面的步骤再创建一个还原点来保护系统。

三、接入外置显卡 并安装驱动；

  1. 接入外置显卡并在启动选择器启动windows
    Mac关机，在左边靠近tab键的雷电3接口插线（其他机器请自行判断哪个接口带宽最好），长按option键选择启动EFI引导启动系统；
    到了下面的界面，按Q键，然后按下⬇️键，选择windows boot启动。最下面的文字 undetected egpu的提示作者说可以忽略。

  1. 在windows安装外置显卡对应的驱动程序
    驱动下载链接地址： AMD Radeon drivers or Nvidia GeForce drivers 自行找到外置显卡最新的驱动安装。
    以我的AORUS GTX 1070 GAMING BOX | Graphics Card - GIGABYTE Global 为例，这时候还要安装几个驱动：GeForce experience（管理驱动的）、外置显卡盒的驱动：AORUS ENGINE 控制显卡的频率
  2. 安装好驱动之后，建议重启后最后制作一次系统还原点。

日常使用windows 启动步骤

论坛的大佬们经过很多的测试之后，以下步骤几乎适合所有的硬件都能够成功启动。启动步骤如下：

  • 关机的状态下，通过制定的雷电3接口插入外置显卡的雷电3连接线，然后开机；
  • 长按option键开启启动引导程序（用rEFInd启动的话开机就会显示启动菜单）
  • 选择引导盘有 apple_set_os.efi 的EFI 启动盘，按回车；
  • 按Q键之后按下键选择win10盘来启动；
  • windows 启动到桌面，右下角显示 外置显卡的标志出现，确认外置显卡激活状态；
  • 大功告成

我的电脑硬件：

  • 2016 13” TouchBar Macbook Pro  - i5 2.9GHz
  • Intel Iris Graphics 550 1536 MB
  • 16GB Ram
  • 258GB SSD
  • Catalina 10.15.1
  • Windows 10 专业版 1909

egpu硬件：

  • Aorus GTX 1070 gaming box
  • 2米长的贝尔金 Thunderbolt 3 cable connected to 左下角 TB3 接口

游玩心得

安装了1070的外置显卡之后，3D大型游戏都玩的相当顺滑。最近登录Steam的荒野大镖客2设置好之后，2K60帧非常顺利，4K有小小卡，也能上。
